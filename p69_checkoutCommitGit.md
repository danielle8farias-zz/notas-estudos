# Desfazendo commits (checkout)

Para voltar ao commit anterior, digite:
```
$ git checkout HEAD~1
```

![voltando ao commit anterior](./img/commitGit2.png)

Caso queira retornar ao último commit, use o comando:
```
$ git checkout master
```
