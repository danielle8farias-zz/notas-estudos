# Tipos de grafos

## Grafo nulo

É aquele que **não possui vértices**.

**G = (V, E)** é nulo se **V = { }** e  **E = { }**.

- **V** número de vértices;
- **E** número de arestas;
- **{ }** conjunto vazio.
